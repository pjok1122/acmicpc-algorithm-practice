# DFS BFS

알고리즘 대회에서 가장 많이 출제되는 유형입니다.

깊이우선 탐색(DFS)은 Stack을 이용하여 모든 노드를 탐색하는 기법이며,

너비우선탐색(BFS)은 Queue를 이용하여 모든 노드를 탐색하는 기법입니다.
